## ЁЯТб Tips for Best Results

### тЬЕ DO:
- **Speak clearly** and at normal pace
- **Hold the button** while speaking (don't release mid-sentence)
- **Use complete sentences** for better recognition
- **Wait for the response** before giving next command
- **Speak in a quiet environment** for best accuracy

### тЭМ DON'T:
- Don't speak too fast or too slow
- Don't release the button while speaking
- Don't use very short commands (< 1 second)
- Don't interrupt the audio response
- Don't use the feature without internet connection

## ЁЯФз Troubleshooting

### Problem: Button is gray
**Solution:** Wait for backend to connect (green cloud icon) or tap cloud icon to retry

### Problem: No response after speaking
**Solution:** Check internet connection and try again

### Problem: Wrong language response
**Solution:** Speak more clearly or update your profile language setting

### Problem: "Recording too short" error
**Solution:** Hold the button longer and speak a complete sentence

### Problem: Stuck in "Processing" state
**Solution:** Tap the button once to force reset
